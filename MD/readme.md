#Proposta
Este projeto foi criado com o intuito de entender mais sobre a linguagem de marcação HTML e também estilização com o CSS, foi uma atividade entregue pelo curso de Desenvolvimento de Sistemas do SENAI, na qual foi nos dada a aoportunidade de criar quaisquer site a partir de nossa própria imaginação.

Tive esta ideia após fazer uma receita de um café artesanal e quis me aprofundar no assunto ao mesmo tempo que fazia um site, nada mais justo do que criar um site baseado na maior franquia de cafés, não? A partir de um planejamento que irei explicar mais pra frente, começei a fazer o projeto, sempre pedindo ajuda quando não entendia algum conceito ou como fazia certa coisa.

Após 2 semanas consegui criar o site base feito com HTML e CSS simples, com tons verdes claros para a estilização, com as seguintes páginas:

##[index.html](https://github.com/ajtelesnavarro/STARBUCKS/blob/master/HTML/index.html)
Começando pelo início, fiz um menu simples com dropdown com os seguintes tópicos: **Menu** e **Pedidos**, nas quais tem os respectivos sites: [bebidas_geladas.html](https://github.com/ajtelesnavarro/STARBUCKS/blob/master/HTML/bebidas_geladas.html), [bebidas_quentes.html](https://github.com/ajtelesnavarro/STARBUCKS/blob/master/HTML/bebidas_quentes.html) e [fazer_pedidos.html](https://github.com/ajtelesnavarro/STARBUCKS/blob/master/HTML/fazer_pedido.html), e que nesta última página também nos leva para [login.html](https://github.com/ajtelesnavarro/STARBUCKS/blob/master/HTML/login.html) que é uma página simples para fazer seu login caso ja o tenha.

Logo abaixo nós temos uma sessão na qual vai ter os produtos mais populares da cafeteira, divididos em 4 bebidas quentes e 4 bebidas frias, e ao final da página também é possível ver um texto do porque o café STARBUCKS™ é o melhor.

##[bebidas_geladas.html](https://github.com/ajtelesnavarro/STARBUCKS/blob/master/HTML/bebidas_geladas.html) e [bebidas_quentes.html](https://github.com/ajtelesnavarro/STARBUCKS/blob/master/HTML/bebidas_quentes.html)
Essas duas páginas são muito parecidas se comparadas lado a lado, o que apenas muda são as bebidas dispostas em cada uma, na qual, assim como toda página destre projeto, possui o mesmo cabeçalho com dropdown e logo abaixo um texto apresentando as bebidas geladas e cada uma das bebidas separadas em containeres espaçados contendo até fontes estilizadas, e é repetido o mesmo processo para ambas as páginas.

##[fazer_pedidos.html](https://github.com/ajtelesnavarro/STARBUCKS/blob/master/HTML/fazer_pedido.html)
Já nesta página, é nos mostrado um formulário personalizado com tons em verde, assim como todo o site, para fazer o seu pedido da sua bebida favorita! Como atualmente não tenho conhecimentos aprofundados em banco de dados e nem em programação back-end, consegui apenas fazer um formulário simples.

Já se o usuário já tiver um login, ele poderia clicar em [login](https://github.com/ajtelesnavarro/STARBUCKS/blob/master/HTML/login.html) que o redicionará para outra página de código simples apenas para fazer seu login já cadastrado.
